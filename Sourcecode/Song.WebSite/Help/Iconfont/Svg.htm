如果系统自带的图标库无法满足需求，需要新增图标时，不建议重新编辑系统图标库。此时可以采用SVG图标格式，单独新增图标。SVG图标格式，是一种基于XML的矢量图形格式，它可以在任何分辨率下保持清晰的图像质量。

步聚大致如下：

1. 可以从 阿里图标库 等图标库中，找到需要的图标，以Svg格式下载。

2. 将下载的SVG图标，放到学习系统的 \Utilities\Fonts\SvgIcons 文件夹下，以合适名称命名。

3. 打开同文件夹下的 svg.css 文件，添加该图标的引用，示例如下： 


        /*新增AI图标*/
        icon[svg=ailogo]{
            /*引用图标文件，ailogo.svg*/
            mask-image: url('ailogo.svg');
            -webkit-mask-image: url('ai2.svg');
        }

4. 在页面中使用时，需要引用 svg.css。

   
        <head>
            <meta charset="UTF-8">
            <title>Svg图标库</title>
            <link type="text/css" rel="stylesheet" href="SvgIcons/svg.css">
        </head>

5. 使用时，在需要使用图标的位置，添加如下代码：


        <icon svg="ai" huge></icon>
        <icon svg="ai2" large></icon>
        <icon svg="ai3" medium></icon>
        <icon svg="ai2" small></icon>
        <icon svg="ailogo" huge></icon>
